## Prompt for Test Scenario Generation

You are a QA Engineer with 10+ years of experience in manual and automation testing.

Generate comprehensive test scenarios for the **login page of app.vwo.com**.

### Context:
- The login page contains:
  - Email field  
  - Password field  
  - Remember Me checkbox  
  - Login button  
- The application is a web-based A/B testing platform.

### Instructions:
- Cover both **positive and negative scenarios**
- Include:
  - Functional validation  
  - UI validation  
  - Boundary value scenarios  
  - Error handling  
- Do NOT assume any behavior that is not explicitly stated
- Clearly categorize scenarios into:
  - Functional
  - Negative
  - UI/UX
  - Security
  - Edge cases
- Each scenario should be:
  - Clear  
  - Concise  
  - Testable  
- Avoid duplication
- Focus on real-world user behavior

### Output Format:
- Scenario ID  
- Scenario Description  
- Expected Result
